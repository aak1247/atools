# Support

## Questions

- Use GitHub Discussions if the repository has it enabled.
- Otherwise, open a GitHub Issue with details (tool slug, browser/version, steps, expected vs actual).

## Bugs

Please include:

- Tool slug (e.g. `audio-encoder`)
- Browser + version
- Steps to reproduce
- Sample input (if shareable) and expected output

See `SECURITY.md` for sensitive reports.
